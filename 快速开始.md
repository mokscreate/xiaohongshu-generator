# 快速开始

## ✅ 项目已创建并运行成功!

开发服务器已启动: **http://localhost:5173**

## 📋 下一步操作

### 1. 配置 API 密钥(必需)

要使用 AI 生成功能,你需要配置 Anthropic API 密钥:

```bash
# 1. 复制环境变量模板
cp .env.example .env

# 2. 编辑 .env 文件,填入你的 API 密钥
# VITE_ANTHROPIC_API_KEY=sk-ant-xxxxx
```

获取 API 密钥: https://console.anthropic.com/

### 2. 重启开发服务器

修改 .env 文件后,需要重启服务器:

```bash
# 停止当前服务器 (Ctrl+C)
# 然后重新启动
npm run dev
```

### 3. 开始使用

在浏览器中打开 http://localhost:5173,按照以下步骤:

1. 选择餐饮方向(饺子馄饨、汉堡薯条等)
2. 选择图片排版方式
3. 选择标题格式
4. 选择正文格式
5. 填写店铺信息
6. 上传图片(可选)
7. 点击生成笔记

## 🛠️ 常用命令

```bash
# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## ⚠️ 注意事项

- 如果没有配置 API 密钥,生成功能将无法使用
- API 调用需要网络连接
- 确保 API 密钥有足够的额度
