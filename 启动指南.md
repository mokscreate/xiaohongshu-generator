# 🚀 启动指南

## ✅ CORS 问题已解决!

现在使用后端代理服务器来调用 MiniMax API,避免了浏览器的跨域限制。

## 📋 启动步骤

### 1. 启动后端服务器(已启动)
```bash
npm run server
```
后端运行在: http://localhost:3001

### 2. 启动前端开发服务器(已启动)
```bash
npm run dev
```
前端运行在: http://localhost:5175

### 3. 使用应用
在浏览器中打开: **http://localhost:5175**

## 🎯 现在可以正常使用了!

1. 刷新浏览器页面
2. 按照步骤填写信息
3. 点击生成笔记
4. AI 将通过后端服务器调用 MiniMax API 生成内容

## 🔧 架构说明

```
浏览器 (localhost:5175)
    ↓
前端 React 应用
    ↓
后端 Express 服务器 (localhost:3001)
    ↓
MiniMax API
```

这样做的好处:
- ✅ 解决了 CORS 跨域问题
- ✅ API 密钥更安全(不暴露在前端)
- ✅ 可以添加更多后端逻辑

## 📝 注意事项

- 需要同时运行前端和后端两个服务
- 如果修改了 .env 文件,需要重启后端服务器
- 查看后端日志可以看到 API 调用详情

## 🛠️ 常用命令

```bash
# 启动后端
npm run server

# 启动前端
npm run dev

# 构建生产版本
npm run build
```
